To reproduce results of GraphBior, folder "signals" and these scripts to the folder of GraphBior.
For details, please see scripts "Biorth_filter_bank_denoise.m" and "Biorth_filter_bank_nla.m".
To obtain GraphBior, download and extract the package at http://perso.ens-lyon.fr/pierre.borgnat/Codes/CoSubFBtoolbox.zip and check the folder of 3rd party packages.
